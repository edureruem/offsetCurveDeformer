# Offset Curve Deformer - Maya 사용자 매뉴얼

## 🎨 **Maya 아티스트를 위한 완벽 가이드**

### **버전**: 1.0.0
### **지원 Maya 버전**: 2022, 2023, 2024, 2025
### **플랫폼**: Windows 10/11, macOS 12+, Linux (CentOS 7+)

---

## 📦 **설치 가이드**

### **방법 1: Maya App Store (권장)**

1. **Maya 실행** → **Windows** → **General Editors** → **App Store**
2. **"Offset Curve Deformer"** 검색
3. **Install** 클릭
4. Maya 재시작

### **방법 2: 수동 설치**

#### **Windows**
```
1. offsetCurveDeformer.mll 파일을 다운로드
2. 파일을 다음 경로에 복사:
   C:\Users\[사용자명]\Documents\maya\[버전]\plug-ins\

3. Maya에서 Plugin Manager 열기:
   Windows → Settings/Preferences → Plug-in Manager

4. offsetCurveDeformer.mll 체크박스 활성화
   ✅ Loaded  ✅ Auto load
```

#### **macOS**
```
1. offsetCurveDeformer.bundle 파일을 다운로드
2. 파일을 다음 경로에 복사:
   ~/Library/Preferences/Autodesk/maya/[버전]/plug-ins/

3. Plugin Manager에서 활성화
```

#### **Linux**
```
1. offsetCurveDeformer.so 파일을 다운로드
2. 파일을 다음 경로에 복사:
   ~/maya/[버전]/plug-ins/

3. Plugin Manager에서 활성화
```

## 🚀 **빠른 시작 가이드**

### **1단계: 기본 설정**

1. **메시 준비**: 변형할 3D 모델 선택
2. **곡선 생성**: 영향을 줄 NURBS 커브 생성
3. **디포머 적용**: 메시 선택 → **Deform** → **Offset Curve Deformer**

### **2단계: 곡선 연결**

```
1. Offset Curve Deformer 노드 선택
2. Attribute Editor에서 "Offset Curves" 섹션 찾기
3. 영향 곡선들을 "Input Curves" 리스트에 추가
   - 곡선 선택 → 마우스 가운데 버튼으로 드래그 → 리스트에 드롭
```

### **3단계: 기본 변형 확인**

- 곡선을 움직여보세요 → 메시가 따라 변형됩니다! ✨

## 🎛️ **상세 사용법**

### **메인 속성 (Main Attributes)**

#### **Offset Mode (오프셋 모드)**
- **Arc Segment**: 빠른 계산 (실시간 작업용) ⚡
- **B-Spline**: 정확한 계산 (최종 렌더링용) 🎯

```
💡 팁: 애니메이션 작업 시에는 Arc Segment 모드로 작업하고,
      최종 렌더링 전에 B-Spline 모드로 전환하세요.
```

#### **Max Influences (최대 영향 수)**
- **범위**: 1-10
- **기본값**: 3
- **설명**: 각 정점이 영향받을 수 있는 최대 곡선 수

#### **Falloff Radius (영향 반경)**
- **범위**: 0.1-100.0
- **기본값**: 5.0
- **설명**: 곡선으로부터의 최대 영향 거리

### **성능 설정 (Performance Settings)**

#### **Use Parallel (병렬 처리)**
- **기본값**: On
- **효과**: CPU 멀티코어 활용으로 2-8배 빠른 계산
- **권장**: 항상 켜두세요 (정점 1000개 이상일 때 효과적)

#### **Use GPU Acceleration (GPU 가속)**
- **조건**: NVIDIA GPU + CUDA 지원
- **효과**: 대용량 메시에서 5-50배 빠른 계산
- **자동 활성화**: 정점 10,000개 이상일 때

## 🎨 **아티스트 제어 (Artist Controls)**

### **Twist (비틀기)**
- **범위**: -360° ~ +360°
- **효과**: 곡선을 따라 메시를 비틀어줍니다
- **수식**: `회전각 = Twist × 곡선파라미터 × 2π`

```
🎯 사용 예: 
- 나무 줄기의 자연스러운 비틀림
- 케이블이나 로프의 꼬임 효과
- 토네이도 같은 회전 변형
```

### **Slide (슬라이딩)**
- **범위**: -100 ~ +100
- **효과**: 곡선을 따라 메시를 앞뒤로 밀어줍니다
- **활용**: 애니메이션 중 메시의 위치 조정

### **Scale (스케일)**
- **범위**: 0.1 ~ 10.0
- **효과**: 곡선을 따라 메시의 크기를 조절합니다
- **Distribution**: 곡선 길이에 따른 스케일 변화 정도

```
🎯 사용 예:
- 뱀의 몸통이 꼬리로 갈수록 가늘어지는 효과
- 나무 가지의 자연스러운 굵기 변화
- 파이프나 호스의 점진적 크기 변화
```

### **Volume Preservation (볼륨 보존)**
- **범위**: 0.0 ~ 1.0
- **효과**: 변형 시 원본 메시의 부피를 보존합니다
- **권장값**: 0.3-0.7 (자연스러운 변형용)

### **Axial Sliding (축 방향 슬라이딩)**
- **범위**: -10 ~ +10
- **효과**: 곡선의 탄젠트 방향으로 추가 이동
- **활용**: 미세한 위치 조정

## 📚 **실무 워크플로우**

### **캐릭터 리깅 워크플로우**

#### **1. 척추/꼬리 리깅**
```
1. 캐릭터 메시 준비
2. 척추를 따라 NURBS 커브 생성
3. Offset Curve Deformer 적용
4. Twist: 10-30° (자연스러운 척추 회전)
5. Volume: 0.5 (근육 볼륨 유지)
```

#### **2. 머리카락/의상 시뮬레이션**
```
1. 머리카락 메시 준비
2. 머리카락 흐름을 따라 가이드 커브 생성
3. Offset Mode: Arc Segment (실시간 시뮬레이션용)
4. Falloff Radius: 2.0 (부드러운 영향)
5. Use Parallel: On (성능 최적화)
```

### **환경 모델링 워크플로우**

#### **1. 유기체 형태 (나무, 산호 등)**
```
1. 기본 형태 메시 생성
2. 성장 방향을 나타내는 커브 생성
3. Scale Distribution: 0.8 (끝으로 갈수록 가늘게)
4. Twist: 15-45° (자연스러운 비틀림)
5. Volume: 0.6 (유기체적 볼륨감)
```

#### **2. 기계/파이프 시스템**
```
1. 파이프 메시와 경로 커브 준비
2. Offset Mode: B-Spline (정확한 형태)
3. Volume: 0.0 (기계적 정확성)
4. Scale: 1.0 (균일한 굵기)
```

## ⚡ **성능 최적화 가이드**

### **메시 크기별 권장 설정**

| 정점 수 | Offset Mode | Use Parallel | 예상 성능 |
|---------|-------------|--------------|-----------|
| < 1,000 | B-Spline | Off | 실시간 |
| 1,000-10,000 | Arc Segment | On | 실시간 |
| 10,000-100,000 | Arc Segment | On + GPU | 거의 실시간 |
| > 100,000 | Arc Segment | On + GPU | 몇 초 지연 |

```

- 다른 오브젝트 숨기기
```

#### **2. 디포머 설정 최적화**
```
- Max Influences: 2-3개로 제한
```

## 🐛 **문제 해결 (Troubleshooting)**

### **일반적인 문제들**

#### **Q: 메시가 변형되지 않아요**
```
A: 체크리스트:
✅ 플러그인이 로드되었는지 확인
✅ 곡선이 Input Curves에 연결되었는지 확인
✅ Falloff Radius가 충분히 큰지 확인
✅ 메시와 곡선이 적절한 거리에 있는지 확인
```

#### **Q: 변형이 너무 느려요**
```
A: 성능 최적화:
⚡ Arc Segment 모드로 전환
⚡ Use Parallel 켜기
⚡ Max Influences 줄이기 (2-3개)
⚡ Falloff Radius 줄이기
⚡ 불필요한 곡선 제거
```

#### **Q: 변형이 부자연스러워요**
```
A: 품질 개선:
🎯 B-Spline 모드로 전환
🎯 Volume Preservation 0.3-0.7로 설정
🎯 Falloff Radius 적절히 조정
🎯 더 많은 곡선으로 영향 분산
🎯 곡선 CV 점 개수 증가
```

#### **Q: Maya가 크래시해요**
```
A: 안정성 확보:
🛡️ Maya 버전 호환성 확인
🛡️ 플러그인 최신 버전 사용
🛡️ 대용량 메시는 단계적으로 적용
🛡️ 다른 플러그인과의 충돌 확인
```

### **에러 메시지 해석**

| 에러 메시지 | 원인 | 해결방법 |
|-------------|------|----------|
| "No influence curves found" | 곡선이 연결되지 않음 | Input Curves에 곡선 추가 |
| "Curve evaluation failed" | 잘못된 곡선 | 곡선 재생성 또는 복구 |
| "GPU acceleration failed" | GPU 드라이버 문제 | 드라이버 업데이트 |
| "Out of memory" | 메모리 부족 | 메시 분할 또는 RAM 증설 |
